Step 1 — Teacher Information & Constraints

Enter each teacher's name, their non-contact entitlement (free periods per week), and any slots when they are unavailable. Teachers defined here are available for assignment to classes in Step 2.

Teachers

Step 2 — Classes & Configuration

Configure the timetable and enter each class to be scheduled. For each class specify the column (A–H), subject, year group, level, how many periods per week it meets, the number of parallel sections, an optional preferred room, and optionally which teacher should take each section. A teacher selected here is a hard rule — that teacher will be required to take that class. Leave on Any teacher for automatic assignment in Step 4.

Classes

Step 3 — Timetable Grid Placement

Click a class in the panel on the right to select it, then click a cell in the grid to place it for that time slot. Each class shows a placed / total counter and its class code(s). Click the × on a placed badge to remove it from that slot.

Step 4 — Automatic Teacher Assignment

Click Run Auto-Assign to match teachers to classes while respecting availability constraints. All class placements from Step 2 are kept exactly as placed. Conflicts are highlighted in red.

Timetable

Step 5 — Manual Conflict Resolution

Drag session cards between grid cells to resolve conflicts. Red highlights indicate a conflict (teacher unavailable or double-booked). Highlights disappear automatically when resolved.

Drag a session card to a different cell to move it.
