Timetable Scheduler

Configuration (JSON)

How to use

Edit the JSON configuration on the left (or use the example), then click Generate Timetable.


Key fields:

  • days_per_week — number of school days
  • periods_per_day — lessons per day
  • teachers — list of teachers with optional unavailable slots
  • rooms — list of rooms with optional capacity
  • columns — option blocks (subjects run in parallel)
  • subjects — each subject has a column, teacher, and optional room
  • unavailable / preferred_periods — use [day, period] pairs (1-indexed)
Configure your timetable and click Generate Timetable to see results.
