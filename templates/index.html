Step 1 — Year Groups & Period Columns

Set up basic timetable settings, define your year groups (e.g. S3, S4, S5) and fill in which column letter each day/period corresponds to for that year group. Not all year groups share the same column at the same time, so each has its own mapping.

Year Groups

Step 2 — Teachers

Enter each teacher's name, their non-contact entitlement (number of free periods per week), and any specific periods when they are unavailable.

Teachers

Step 3 — Classes

Enter each class to be scheduled. Select the year group, subject, level, and column — the number of periods per week is filled in automatically from the period map in Step 1. If a class has multiple sections each section gets its own code (e.g. S4-P-H-E-1 and S4-P-H-E-2). Use the per-section teacher dropdowns to assign teachers: leave a section on Any teacher for automatic assignment. A hard rule is applied only when all sections have a specific teacher selected (1 section with 1 teacher, or 2 sections each with a different teacher).

Classes

Step 4 — Results & Review

Complete all three steps above and click Generate Timetable to see results.
